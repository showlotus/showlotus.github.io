Mac 使用手册

  • 快捷键：https://support.apple.com/zh-cn/HT201236
  • 触摸板手势：https://support.apple.com/zh-cn/HT204895

安装 brew

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

终端配置

  • 安装 iTerm2：https://iterm2.com/

  • oh-my-zsh：

    • 安装

      • 方式一：

        sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

# 国内镜像
sh -c "$(curl -fsSL https://gitee.com/mirrors/oh-my-zsh/raw/master/tools/install.sh)"

      • 方式二：

        从 GitHub 克隆，然后进入到 tools 目录下执行下面的命令：

        sh install.sh

    • 主题

      可选主题：https://github.com/ohmyzsh/ohmyzsh/wiki/Themes

      切换主题：

      1. 打开配置文件

        vim ~/.zshrc

      2. 更改为对应的主题名

        ZSH_THEME="robbyrussell"

      3. 保存并退出

        :wq

    • 安装 powerlevel10k，下载字体。如果重新配置，输入 p10k configure。

      贴一下配置后的 iTerm2 的样子：

  • 语法高亮：zsh-syntax-highlighting

  • 命令补全提示：zsh-autosuggestions

NodeJS

  • 安装

    brew install node

  • 安装版本切换工具 n

    • 安装

      npm i n -g

    • 使用

      • 安装指定版本，版本号不带 v 前缀

        n [版本号]

        如果安装报错没权限，使用 sudo 命令

        sudo n [版本号]

        如果安装版本超时，使用阿里镜像（贼快～）

        sudo NODE_MIRROR=http://npm.taobao.org/mirrors/node n [版本号]

      • 切换版本

        n

        然后出现一个界面，按照提示操作即可

Git

  • 安装

    brew install git

  • 打开 .gitconfig 文件

    open ~/.gitconfig

  • 全局忽略 .DS_Store

    新建全局忽略配置文件 .gitignore_global

    touch ~/.gitignore_global

    写入内容

    .DS_Store
**/.DS_Store
.DS_Store?

    全局配置

    git config --global core.excludesfile ~/.gitignore_global

    删除仓库已有的 .DS_Store 文件

    find . -name .DS_Store -print0 | xargs -0 git rm -f --ignore-unmatch

  • 显示隐藏文件快捷键：⌘ + Shift + .

  • 生成 SSH

    • 打开终端执行下方命令：

      ssh-keygen -t rsa -C "xxx@qq.com"

      紧接着会出现提示，默认创建在 /Users/xxx/.ssh/id_rsa 路径下，然后输入两次密码

      Generating public/private rsa key pair.
Enter file in which to save the key (/Users/xxx/.ssh/id_rsa):
Created directory '/Users/xxx/.ssh'.
Enter passphrase (empty for no passphrase):
Enter same passphrase again:

    • 打开 .ssh/id_rsa.pub 文件

      cat .ssh/id_rsa.pub

      复制内容，然后再在 GitHub -> Settings -> SSH and GPG keys 里新建 SSH key。

    • 验证

      ssh -T git@github.com

      出现下面的内容，则表示连接成功。

      Hi xxx! You've successfully authenticated, but GitHub does not provide shell access.
