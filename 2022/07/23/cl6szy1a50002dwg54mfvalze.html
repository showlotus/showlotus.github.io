JavaScript

this指向

2022-07-23

问题

最近在研究 Vue 中 provide 和 inject 原理的时候，本来是打算，看看能否解决之前遇见的一个疑惑：

  • 父组件

    // parent
<template>
  <div>
    <input v-model="msg"></input>
  </div>
</template>

<script>
export default {
  provide() {
    return {
      getMsg: () => this.msg,
      getMsg2() { return this.msg },
      getMsg3: this.getMsg3,
    };
  },
  data() {
    return {
      msg: "parent msg",
    };
  },
  methods: {
    getMsg3() {
      return this.msg;
    }
  }
};
</script>

  • 子组件

    // child
<template>
  <div>
    <p>getMsg: {{ getMsg() }}</p>
    <p>getMsg2: {{ getMsg2() }}</p>
    <p>getMsg3: {{ getMsg3() }}</p>
  </div>
</template>

<script>
export default {
  inject: ["getMsg", "getMsg2", "getMsg3"],
  data() {
    return {
      msg: "child msg"
    }
  }
}
</script>

子组件中的 getMsg 和 getMsg3 得到的都是 parent msg 并且都是响应式的，而 getMsg2 得到的是 child msg。大致有了思路：这是一个关于 this 指向的问题。

案例

先来看一个类似的案例：

案例一

const obj = {
  name: "obj",
  getName() {
    return this.name;
  },
};

const provide = function () {
  return {
    getName: () => this.name,
    getName2() {
      return this.name;
    },
    getName3: this.getName,
  };
};

const inject = provide.call(obj);

console.log("getName:", inject.getName()); // "obj"
console.log("getName2:", inject.getName2()); // undefined
console.log("getName3:", inject.getName3()); // undefined

getName 的 this 指向 obj，那 getName2 和 getName3 的 this 指向哪?

案例二

const obj = {
  name: "obj",
  getName() {
    return this.name;
  },
};

const provide = function () {
  return {
    getName: () => this.name,
    getName2() {
      return this.name;
    },
    getName3: this.getName.bind(obj), // <= 使用 bind 修改 this
  };
};

const inject = provide.call(obj);

console.log("getName:", inject.getName()); // "obj"
console.log("getName2:", inject.getName2()); // undefined
console.log("getName3:", inject.getName3()); // "obj"

getName3 修改 this 为 obj 后，毫无疑问，打印的结果是 "obj"。

案例三

const obj = {
  name: "obj",
  getName() {
    return this.name;
  },
};

const provide = function () {
  return {
    name: "provide", // <= 在 provide 中添加一个 name 属性
    getName: () => this.name,
    getName2() {
      return this.name;
    },
    getName3: this.getName.bind(obj),
  };
};

const inject = provide.call(obj);

console.log("getName:", inject.getName()); // "obj"
console.log("getName2:", inject.getName2()); // "provide"
console.log("getName3:", inject.getName3()); // "obj"

getName2 的 this 指向 provide。
